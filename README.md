# GamingWebsite

#Vortex Valour – Interactive Gaming Website

Vortex Valour is a futuristic, interactive gaming website designed with a strong sci-fi aesthetic, smooth gradient visuals, and immersive UI elements. The site delivers a high-energy gaming experience focused on conquest, competition, and community engagement.

Built using HTML, CSS, JavaScript, and PHP, the project combines modern front-end design with basic back-end functionality and runs locally using XAMPP.

#Features

Dynamic gradient-based user interface with a cyberpunk theme

Interactive elements and smooth visual transitions

Clean navigation with multiple sections (Home, Battle, About, Music, and more)

Call-to-action components such as Subscribe and Play Now

Social media integration

Responsive layout optimized for different screen sizes

#Tech Stack

HTML5 – Structure

CSS3 – Styling, gradients, and animations

JavaScript – Interactivity and UI behavior

PHP – Server-side logic

XAMPP – Local development environment

#Purpose

This project was created to practice and showcase:

Front-end UI/UX design

Gradient color design and modern gaming aesthetics

Interactive web components

Integration between front-end and back-end using PHP

#Getting Started

Install XAMPP

Place the project folder inside the htdocs directory

Start Apache from the XAMPP control panel

Open localhost/your-folder-name in your browser
